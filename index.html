DropCloud

Your private One-Click uploader for OwnCloud, build on and for Mac OS X.

Requirements

  • Mac OS X 10.8 or later

Development

DropCloud uses CocoaPods to handle it's dependencies and the git-flow branching model. To build DropCloud, you have to install all dependencies first (pod install). Make sure to open the DropCloud.xcworkspace, instead of the .xcodeproj.
