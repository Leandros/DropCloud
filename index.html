ownDrop

Your private one-click uploader for ownCloud, built on and for Mac OS X.

Installation

Download the current version here. Unzip the file, move ownDrop.app to the Application folder and start it.

Before first use you have to fill in your ownCloud server and your credentials. Just click on the icon in the menu bar and go to the settings. Below you can see an example:

Requirements

  • Mac OS X 10.8 or later

Development

ownDrop uses CocoaPods to handle its dependencies and the git-flow branching model. The master branch is always stable and the develop branch is the version, which is currently in development. To build ownDrop you have to install all dependencies first (pod install). Make sure to open the DropCloud.xcworkspace, instead of the .xcodeproj.
